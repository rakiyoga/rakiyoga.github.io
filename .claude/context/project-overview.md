# Project Overview — Raki Yoga

## About
Raki Yoga is the personal website and professional platform for **Rakesh Singh ("Raki")**, an Ayurvedic health consultant based in **Berlin, Germany**. The site serves as an educational resource for Ayurveda, yoga, and holistic wellness, as well as a portal for consultations.

## Domain Knowledge

### Ayurveda
- Ancient Indian system of holistic medicine (5,000+ years old)
- Based on balance of three doshas: **Vata** (air/ether), **Pitta** (fire/water), **Kapha** (earth/water)
- Key texts: Charaka Samhita, Sushruta Samhita, Ashtanga Hridayam
- Incorporates diet (Ahara), lifestyle (Vihara), and herbal remedies (Dravyaguna)

### Raki's Specialization
- 10+ years of Ayurvedic practice
- Trained under Dr. Dilip Prabhakar Gadgil (Tilak Maharashtra Vidyapeeth)
- Focus areas: autoimmune disorders, cancer, neurological conditions
- Approach: natural healing, freedom from medications

### Website Purpose
1. **Educate**: Ayurveda fundamentals, blog articles, recipes
2. **Consult**: In-person (Berlin) and online consultations
3. **Connect**: Social media presence, contact form, community building

## Architecture Decisions
1. **Jekyll + GitHub Pages**: Chosen for simplicity, cost-effectiveness, and fast static hosting
2. **Beautiful Jekyll theme**: Provides responsive design, blog support, and customization without heavy development
3. **Markdown content**: Easy for non-developers to edit content
4. **Green color theme**: Represents nature, health, and Ayurvedic philosophy
5. **Expandable sections**: `<details>` elements manage dense educational content
6. **Numbered page files**: Simple navbar ordering mechanism
